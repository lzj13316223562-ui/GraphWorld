# T-RO LaTeX Notes

This directory is a writing workspace for an IEEE Transactions on Robotics style manuscript.

Current setup:

- `main.tex`: IEEE journal-style manuscript skeleton for GraphWorld
- `refs.bib`: starter bibliography

Template choice:

- IEEE's journal author center says to use the IEEE article templates in Word or LaTeX.
- The IEEE article template page points authors to the IEEE Template Selector for publication-specific templates.
- Overleaf also provides the standard IEEE journal skeleton based on `IEEEtran.cls`.

Practical note:

- This machine currently does not have `pdflatex`, `bibtex`, or `latexmk` installed, so the files were prepared but not compiled locally.
