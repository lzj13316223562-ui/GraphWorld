# Related Works 精读笔记

这组笔记服务于 GraphWorld 的论文写作与 related works 梳理，重点不是做“百科式摘要”，而是紧扣我们自己的研究问题来读：

- 如何支持持续演化的开放环境
- 如何支持图结构世界底座
- 如何支持长期、非单轮的任务评测
- 如何支持人类活动、社会交互与外生事件
- 如何支持 state-aware / closed-loop / graph-grounded 的 agent 决策
